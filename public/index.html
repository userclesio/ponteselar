🔐 Segurança do Endpoint

Segredo enviado no header Authorization: Bearer … para proteger /api/sync de acesso não autorizado.

Deve ser igual à variável de ambiente na Vercel.

🚀 Deploy na Vercel

Passos para colocar a integração em produção.

  1. Suba este repositório no GitHub (ou GitLab/Bitbucket).
  2. Acesse vercel.com → New Project → importe o repo.
  3. Em Settings → Environment Variables adicione as três chaves abaixo.
  4. O cron em vercel.json já dispara /api/sync a cada 5 min automaticamente.
  5. (Recomendado) Crie um Vercel KV store e conecte ao projeto para deduplicação persistente.

📋 Variáveis de Ambiente

Copie e cole no arquivo .env.local para testes locais, ou use como referência ao configurar na Vercel. Nunca commite este arquivo com valores reais.

# Selar
SELAR_API_KEY=sua_chave_selar_aqui

# UTMify
UTMIFY_API_TOKEN=seu_token_utmify_aqui

# Segurança do endpoint /api/sync
CRON_SECRET=seu_segredo_aqui
